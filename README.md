# Health Tracker Artefact

This repository contains the artefact submission for my Health Tracker project.

## Included files
- Weillo.zip — complete project files
- Weillo.sql — database export

## Project type
PHP / MySQL web application

## How to review
1. Download the ZIP file and extract it.
2. Place the project in a local PHP server environment such as XAMPP.
3. Import the SQL file into phpMyAdmin / MySQL.
4. Open the project through the local server.

## Main project structure
- config/
- controllers/
- public/

This repository is provided for assessment of the project artefact and supporting components.
